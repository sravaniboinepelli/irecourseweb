Preprocessing:

  1. Removed noisy data like emoticons and non-ASCII data and single character hashtags
  2. Replaced numbers, URLs, tagged users etc with corresponding tags (DIGIT, MENTION, URL etc).
  3. Segregate by tag, post and comments for overall context
  4. Handling multiple hashtags: post 1 h1 con1, post 1 h2 con2 etc etc (each hashtag should correspond to the same post)
  5. For training instances tagged with multiple gold-standard hashtags, we copy the instances multiple times, each with one gold standard hashtag following.
  6. Splitting hashtag for better context and accuracy using the ekphrasis which uses Viterbi algorithm

Methodologies tried:

  1. The State-of-the-art model uses Sequence 2 Sequence with Bi-attention mechanism network which takes posts as well as conversations as data and its corresponding hashtags as true label.
  2. We tried clustering and categorizing the generated hashtags to further incorporate ranking by doing topic modelling as well but training the LDA performed poorly as most of the words were named entities and the size of the post was less resulting in bad accuracy.
